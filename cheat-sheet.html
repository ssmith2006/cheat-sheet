Cheat Sheet Commands

Git Commands

git init: to initialize a new Git repository in current directory.

git clone: clone a remote repo to the local machine.

git add: to add a file

git commit -m: to commit changes

git push: to push changes to the remote repo.

Bash Commands

cd: change directory.

ls: list files that are located in the current directory

touch: to create an empty file

mkdir: to make a new directory

pwd: shows what is in the current directory.
